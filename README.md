# Goland vs Rust 性能测试

这个测试是计算规划求解问题。
一共要执行 `2^30`个循环，每个循环还要依次判断本次循环数的二进制位中的1。

本测试是使用了多线程cpu密集型计算的设计方法，给每一个物理cpu内核都分配一个线程，并且平均分配所有的循环。
